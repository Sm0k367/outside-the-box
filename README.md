## 🚀 How to Wake the Swarm

```bash
git clone https://github.com/Sm0k367/outside-the-box.git
cd outside-the-box

# Add your Groq key
cp .env.example .env.local
# Add GROQ_API_KEY=your_key_here to .env.local

npm install
npm run dev
```

Then open **http://localhost:3000** and talk to the swarm.

The agents will:
- Read their own `.md` files for instructions
- Use real tools (they can modify this repository if you ask)
- Coordinate with each other through the orchestrator
- Give you production-ready output
