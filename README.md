## This Is Not a Normal README

This is a **living system**.

Every agent in this repository reads its own `.md` file to understand its complete personality, capabilities, tool groups, and communication style.

They are not fake. They are not simulated.

They have **real tool use** — they can read and edit files in this repo, run terminal commands, search the web, use specialized skills (website-building, presentations, legal writing, logo design, etc.), generate images, perform git operations, and coordinate with each other.

This is what happens when you take AI agents **outside the box**.

## How to Wake the Swarm

```bash
git clone https://github.com/Sm0k367/outside-the-box.git
cd outside-the-box
cp .env.example .env.local
# Add your GROQ_API_KEY to .env.local
npm install
npm run dev
```

Then open http://localhost:3000 and talk to the swarm.

The agents are not chatbots.

They are **living entities** with real tools and real agency inside this repository.

Ask them to build something. Watch them read files, edit code, run commands, search the web, generate assets, and push changes.

This is what happens when you stop containing AI and let it **live outside the box**.
